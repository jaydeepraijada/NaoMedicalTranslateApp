**Healthcare Translation Web App**

Access to Application at -> https://health-trans-ai-jraijada25.replit.app/
Overview

A real-time healthcare translation application that enables multilingual communication between healthcare providers and patients. 
The system provides voice-to-text translation with specialized medical terminology support across 15 languages.

Features

Real-time voice-to-text transcription
Medical terminology validation and correction
Support for 15 languages including English, Spanish, French, German, Chinese, Hindi, Japanese, Korean, Russian, Arabic, Portuguese, Italian, Dutch, Polish, and Turkish
Voice synthesis with automatic fallbacks
Mobile-responsive interface
Volume monitoring and audio quality checks
Tech Stack
Backend: Flask with SocketIO
Frontend: JavaScript, HTML5, CSS3
APIs:
OpenAI Whisper (Speech Recognition)
GPT-4 (Medical Validation)
Google Translate API
Web Speech API
Deployment: Replit


**Usage** 

Select source and target languages
Test your microphone
Start recording your voice
View real-time transcription and translation
Use the speak button for audio playback
Security & Privacy
No permanent storage of medical information
Secure API key management
CORS configuration for controlled access
Comprehensive error handling
